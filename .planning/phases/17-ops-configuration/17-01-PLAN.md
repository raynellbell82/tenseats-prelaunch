Create a new webhook endpoint in the Stripe Dashboard that points to the Convex billing webhook handler.

**Steps:**
1. Open Stripe Dashboard -> Developers -> Webhooks
2. Click "Add endpoint"
3. Set endpoint URL: `https://api.tenseats.io/stripe/billing-webhook`
4. Select events to listen for (these are the events the @convex-dev/stripe component handles):
   - `customer.subscription.created`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
   - `invoice.payment_succeeded`
   - `invoice.payment_failed`
   - `checkout.session.completed`
5. Click "Add endpoint"
6. After creation, click the endpoint -> "Signing secret" -> "Reveal" to get the signing secret
7. Copy this signing secret — it is the value for `STRIPE_BILLING_WEBHOOK_SECRET` in Convex (Task 1, Variable 2)
8. Go back to Convex Dashboard and set `STRIPE_BILLING_WEBHOOK_SECRET` to this value

**IMPORTANT:** The endpoint URL must be exactly `https://api.tenseats.io/stripe/billing-webhook` — this matches the `webhookPath` in `convex/http.ts` line 38. The Convex deployment serves HTTP actions at `api.tenseats.io`.
