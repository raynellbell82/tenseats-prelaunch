# Phase 24: Vendor Success Page - Context

**Gathered:** 2026-03-14
**Status:** Ready for planning

<domain>
## Phase Boundary

After OTP verification, vendor-role users (Chef, Mixologist, Venue Host, Creator) see a dedicated success page at `/launch/success/vendor` with role-specific welcome messaging, optional Stripe Express Connect step, Zoho One recommendation, and social links. Replaces the existing stub page. Uses the PostSignupLayout, VerticalTimeline, and SocialLinks from Phases 22/23.

</domain>

<decisions>
## Implementation Decisions

### Vendor welcome messaging
- Role-themed metaphor headlines (not generic vendor message):
  - Chef: "The kitchen is yours." (or similar — copy-writer skill generates)
  - Mixologist: "The bar is set." (or similar)
  - Creator: "The spotlight is on." (or similar)
  - Venue Host: "The doors are open." (or similar)
- Headline only is role-specific — body copy is shared across all vendor roles
- Body copy tone: community + action blend — welcome to the community AND here's how to prepare
- Copy generated by tenseats-copy-writer skill, reviewed at checkpoint
- Green check circle with pulse reused (consistent with verify and guest pages)
- Need to query user's role from userRoles table to display the correct headline

### Stripe Connect presentation
- Timeline step with inline action button — integrated into VerticalTimeline, not a standalone section
- Low-pressure optional: "You can connect Stripe now or wait until launch" — no urgency, subtle/outline button style
- Real Stripe Express Connect onboarding — button initiates actual Stripe Express flow, not a placeholder
- Timeline step is active (green dot) even if not yet connected

### Zoho One recommendation
- Info card below the VerticalTimeline — separate visual block with more weight than a timeline step
- Partner endorsement tone: "We recommend Zoho One" — official Tenseats recommendation
- Tagline refined by copy-writer skill (based on "one fee for 50+ best-in-class tools" core value prop)
- Link to `https://go.zoho.com/Slvq` opens in new tab

### Page structure
- Section order: Headline → Green check → VerticalTimeline → Zoho card → Social links
- Same structure as guest page with Zoho card inserted before social
- Timeline steps: 1) Connect Stripe Express (optional, with button) 2) Add supportteams@tenseats.io to contacts 3) Explore soon teaser
- Click-to-copy for support email — same pattern as guest page
- Auth guard: only OTP-verified users with vendor role can access — redirect to `/launch` otherwise
- SocialLinks component imported directly from Phase 23's shared component

### Claude's Discretion
- Exact role-themed headlines (copy-writer generates options)
- Body copy length and wording
- Zoho card visual design (border, background, icon choice)
- Stripe button styling within the timeline
- Spacing and layout details

</decisions>
