# Phase 8.1: Cities Globe - Context

**Gathered:** 2026-03-13
**Status:** Ready for planning
**Source:** PRD Express Path (docs/superpowers/specs/2026-03-13-cities-globe-section-design.md)

<domain>
## Phase Boundary

Add an interactive 3D globe section to the homepage that shows Tenseats' 32 US city markets as glowing markers with region-tabbed city pills for discovery and navigation. Uses `cobe` (~5KB WebGL globe), existing METROS_DATA coordinates, and Framer Motion for animations.

</domain>

<decisions>
## Implementation Decisions

### Component Architecture
- Single client component `CitiesGlobe` at `components/landing/cities-globe.tsx`
- Rendered from `app/page.tsx` directly below `HeroSection`
- No wrapper divs at page level — component manages its own layout

### Globe Library
- Use `cobe` (~5KB) — lightweight WebGL globe, no API key, no external tiles
- Dynamically imported with `next/dynamic` + `ssr: false`
- `createGlobe(canvasRef.current!, { ... })` inside `useEffect` with `globe.destroy()` cleanup
- Initial `phi` calculated for continental US center (~longitude -98°): `((98 + 180) * Math.PI) / 180 - Math.PI`
- `devicePixelRatio: Math.min(window.devicePixelRatio, 2)`

### Globe Appearance
- Dark globe with subtle land mass shading
- All 32 cities as equal-size orange glowing dots (`#f97316` / `[0.976, 0.451, 0.086]`)
- `dark: 1`, `diffuse: 1.2`, `mapSamples: 16000`, `mapBrightness: 2`
- `baseColor: [0.1, 0.1, 0.12]`, `glowColor: [0.05, 0.05, 0.08]`

### Globe Behavior
- Auto-rotates slowly when idle, starts focused on continental US
- Pill hover: globe smoothly rotates to center hovered city (lerp in onRender), dot pulses, tooltip appears
- Idle return: resumes auto-rotation after ~2s of no hover
- Touch devices: tap-to-navigate pills only, globe auto-rotates continuously
- `prefers-reduced-motion`: disable auto-rotation, instant position changes

### Section Header
- "Now in 32 Cities" eyebrow (orange, uppercase, small)
- "Find Your City" headline
- Subtitle: "Exclusive dining experiences are happening near you. Explore your city's hidden culinary scene."

### Region Tabs
- Horizontally centered pill-shaped tabs: South, Midwest, Northeast, West
- Active: orange background tint + orange text + orange border
- Inactive: subtle border, muted text
- Framer Motion `AnimatePresence` for pill list transitions

### City Pills
- Wrapped flex layout, centered, gap between pills
- Each pill is a Next.js `<Link>` to `/cities/[slug]`
- Default: dark background, subtle border, light text
- Hover: orange border, brighter text, globe rotates to city
- Hover debounced at 150ms to prevent rapid rotation changes
- Framer Motion `layout` animation for reflow on tab change

### Region Mapping
- New file `lib/city-regions.ts` with `CITY_REGIONS` constant mapping slug → region
- Regions: South, Midwest, Northeast, West
- Oklahoma City classified as South (US Census Bureau convention)

### Data Source
- `METROS_DATA` from `lib/city-data.ts` (static copy, not re-export from convex)
- Coordinates used for cobe markers, displayName for pill labels

### WebGL Fallback
- Wrap `createGlobe` in try/catch
- On failure, set `globeReady` state to false, hide canvas
- Tabs + pills remain fully functional without globe

### Performance
- `cobe` dynamically imported (no SSR)
- IntersectionObserver pauses globe rendering when off-screen
- `will-change: transform` on canvas for GPU compositing
- Cleanup on unmount / route change via destroy function

### Accessibility
- Globe canvas: `aria-hidden="true"`, wrapper with `role="img"` + alt text
- Region tabs: `role="tablist"` / `role="tab"` / `aria-selected`
- City pills: standard `<a>` links, keyboard navigable
- `prefers-reduced-motion` honored

### Responsive
- Desktop (≥1024px): 500×500 canvas, horizontal tab row, wrapped pills ~3 rows
- Tablet (768-1023px): 400×400 canvas, horizontal tab row, wrapped pills ~4 rows
- Mobile (<768px): 300×300 canvas, horizontal scroll tabs, full-width pills

### Claude's Discretion
- Exact animation easing curves and durations
- Tooltip positioning and styling details
- Exact padding/margin values within the section
- Whether to add a subtle "Explore all cities" link

</decisions>
