## Decisions Made

- **cities-globe-loader.tsx client wrapper:** Next.js 16 with Turbopack enforces that `next/dynamic` with `{ ssr: false }` must be called from a Client Component. The plan assumed this could live in the Server Component `page.tsx`, but the build failed. Fixed by creating a thin `"use client"` loader file.
- **Actual city list differs from plan examples:** The plan's region list referenced cities like miami-fl, nashville-tn, boston-ma, san-francisco-ca etc. that do not exist in METROS_DATA. Mapped the actual 32 METROS_DATA slugs using US Census Bureau conventions. Washington DC classified as South.
- **South has 16 cities, West only 3:** Reflects actual Tenseats market distribution (heavy South/Midwest presence, lighter West).
