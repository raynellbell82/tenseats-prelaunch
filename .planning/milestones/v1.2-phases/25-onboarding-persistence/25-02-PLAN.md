**Update the stripeAction JSX:**

Replace the current `stripeAction` const with conditional rendering:

```typescript
const isStripeConnected = stripeStatus?.accountId != null;

const stripeAction = isStripeConnected ? (
  // Connected state: green checkmark + label
  <div className="flex items-center gap-1.5">
    <CheckCircle2 className="h-4 w-4 text-green-500" />
    <span className="text-xs text-green-600 dark:text-green-400 font-medium">

    </span>
  </div>

  // Not connected: original button
  <div className="flex flex-col items-start gap-1">

      onClick={handleStripeConnect}

    >
      {connecting ? "Connecting..." : "Connect Stripe"}
    </button>

    </span>
```

Note: `CheckCircle2` is already imported (used for the hero check icon). No new imports needed for the icon.

**Key behaviors:**
- When `stripeStatus.accountId` is non-null (account saved in Convex): show green checkmark + "Stripe Connected"
- When `stripeStatus` is undefined (loading) or accountId is null: show the original Connect button
- The `useQuery` is reactive — when the webhook writes `stripeConnectComplete: true` to the user record, the query automatically re-fires and the UI updates without refresh
- Do NOT change any other part of the page (timeline steps, Zoho card, social links, copy text)
- Do NOT update `isProfileComplete` anywhere (per user decision)

**Handle return from Stripe:**
The return_url from Plan 01 includes `?stripe_connect=complete` query param. No special handling needed — the reactive query will pick up the saved account ID automatically. But clean up the URL param to avoid confusion:

```typescript
import { useSearchParams } from "next/navigation";

// Inside component:
const searchParams = useSearchParams();

useEffect(() => {
  // Clean up stripe_connect query param after return from Stripe
  if (searchParams.get("stripe_connect") === "complete") {
    const url = new URL(window.location.href);
    url.searchParams.delete("stripe_connect");
    window.history.replaceState({}, "", url.pathname);
  }
}, [searchParams]);
```
